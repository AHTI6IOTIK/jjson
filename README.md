# J.JSON

A CLI json beautify program written by node.js.

## Installation

```shell
$ npm install jjson -g
```

## Usage

```
Usage: jjson [options]

Options:
   -f, --file       JSON filename.
   -e, --encoding   JSON file encoding.  [default to utf8]
   -i, --indent     Number of indent for each line.  [default to 2]
```
